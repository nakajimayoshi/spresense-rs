# `cxd56-pac`

This is a [Peripheral Access Crate](https://rust-embedded.github.io/book/start/registers.html) for Sony cxd56 microcontrollers.

This crate is a meta-crate containing PACs created using different tools (svd2rust, chiptools, svd2pac, etc.). Since I'm still exploring what each tool provides this provides a (hopefully) helpful way to try the different crates and organize it into eventually a single PAC later on.
